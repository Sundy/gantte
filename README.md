# GEO Project Plan - Interactive Gantt Chart

这是一个基于 React + Vite 开发的交互式甘特图应用，支持任务管理、层级展示、自动编号以及本地数据持久化。

## 主要功能

- **任务管理**：支持添加、编辑、删除任务，设置任务颜色、起止时间及持续时长。
- **层级结构**：支持无限层级的子任务嵌套，可折叠/展开父任务。
- **自动编号**：根据层级结构自动生成任务编号（如 1, 1.1, 1.1.1），界面整洁对齐。
- **动态时间轴**：
  - 可配置项目开始日期和总持续周数。
  - 自动计算并显示具体的周日期（如 12/08）。
  - 支持固定宽度的周视图。
- **本地数据持久化**：
  - 前端操作自动同步保存到本地 `gantt_data.json` 文件。
  - 包含任务列表、项目开始日期及持续周数配置。
- **可视化优化**：
  - 一级任务红色高亮。
  - 长文本任务名称自动换行对齐。
  - 时间轴进度条垂直居中。

## 快速开始

### 1. 安装依赖

```bash
npm install
```

### 2. 启动服务

本项目包含前端应用和后端 API 服务（用于读写本地文件）。

**启动后端服务 (端口 3001)**
用于处理 `gantt_data.json` 的读写。

```bash
node server.js
```

**启动前端应用 (端口 5173)**

```bash
npm run dev
```

打开浏览器访问终端输出的地址（通常是 http://localhost:5173）。

## 数据存储

所有数据会自动保存到项目根目录下的 `gantt_data.json` 文件中。
该文件包含：
- `tasks`: 任务列表数组
- `projectStartDate`: 项目开始日期
- `projectDurationWeeks`: 项目持续周数

## 技术栈

- React 18
- Vite
- Express (用于本地文件 I/O)
- CSS Grid 布局
